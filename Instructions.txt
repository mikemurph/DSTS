#INSTALLS (pip for MAC, conda for WINDOWS)
- install flask
- install sqlite3 (conda install -c blaze sqlite3=3.8.6)

#RUNNING
1) python app.py
2) you will see something along the lines of "Running on http://"
3) Copy/paste the "http://"" link (localhost) into your browser.
